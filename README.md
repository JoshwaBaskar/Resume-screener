# 🤖 AI Resume Screening Agent

An AI-powered Resume Screening Agent that analyzes resumes against job descriptions, calculates ATS compatibility, identifies skill gaps, generates interview questions, and provides career recommendations.

> 🚀 Built using React, TypeScript, Node.js, Express, and Generative AI APIs.

---

## 📌 Features

### 📄 Resume Upload
- Upload resumes in TXT, PDF, or DOCX format
- Resume parsing and text extraction
- Resume preview

### 🎯 ATS Resume Analysis
- ATS compatibility score
- Resume quality evaluation
- Section-wise analysis
- Keyword matching

### 🧠 Skill Gap Analysis
- Compare candidate skills with job requirements
- Missing skills detection
- Skill recommendations
- Technical and soft skill evaluation

### 💼 Job Matching
- Match resumes against job descriptions
- Candidate ranking
- Match percentage
- Best-fit role suggestions

### 🎤 AI Interview Generator
- Technical interview questions
- HR interview questions
- Behavioral interview questions
- Difficulty-based questions

### 📈 Career Insights
- Career roadmap
- Learning recommendations
- Trending technologies
- Improvement suggestions

### 📝 AI Cover Letter Generator
- Personalized cover letter generation
- Job-specific customization

### 📊 Dashboard
- Resume statistics
- Analysis reports
- Downloadable reports
- Interactive charts

# 🚀 Installation

Clone the repository

```bash
git clone https://github.com/YourUsername/resume-screening-agent.git
```

Go to the project

```bash
cd resume-screening-agent
```

Install dependencies

```bash
npm install
```

Run the frontend

```bash
npm run dev
```

Run the backend

```bash
npm run server
```

---

# ⚙️ Environment Variables

Create a `.env` file in the project root.

```env
GEMINI_API_KEY=YOUR_API_KEY

# OR

OPENAI_API_KEY=YOUR_API_KEY
```
